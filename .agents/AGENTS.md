# Canonical Entigram Governance

Before working in this repository, read `.etg/agent_policy.md` and follow it as
the canonical Entigram governance policy. That policy defines the boot sequence,
impact analysis expectation, pre-handoff gate, Warden ordering, and delivery
snapshot rules. If another instruction file conflicts with `.etg/agent_policy.md`,
the canonical policy wins.
