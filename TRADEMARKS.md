# Entigram Trademark Policy

This policy separates Entigram's open-source code license from use of the
Entigram name, certification language, and related marks.

## Allowed Use

You may use the Entigram name to truthfully describe compatibility, origin, or
dependency relationships, for example:

- "built with Entigram"
- "compatible with Entigram"
- "uses the Entigram package catalog format"

This is nominative use. It must not imply endorsement, certification,
sponsorship, or ownership by the Entigram project.

## Restricted Use

You may not use Entigram marks in a way that suggests an unofficial product,
service, fork, registry, package, or hosted offering is operated by, endorsed
by, or certified by Entigram.

Restricted phrases include, unless explicitly authorized:

- Entigram Certified
- Entigram Registry
- Entigram Cloud
- Official Entigram Package
- Certified Entigram Package

## Package Certification

The `Entigram Certified` designation is reserved for packages, registries, and
services that pass the current Entigram certification process and carry signed
catalog provenance. Forks and third-party packages may describe compatibility,
but they may not claim certification without approval.

## Open Source License

Entigram source code is licensed under Apache-2.0 unless a file or package says
otherwise. That license grants copyright and patent permissions for covered
code, but it does not grant trademark rights.

## Reporting Misuse

Report confusing or misleading use of Entigram marks to the project owner.
